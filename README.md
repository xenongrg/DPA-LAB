Introduction to R Programming:
What is R and RStudio?
R is a statistical software program. It has extremely useful tools for data exploration, data
analysis, and data visualization. It is flexible and also allows for advanced programming. RStudio
is a user interface for R, which provides a nice environment for working with R.
